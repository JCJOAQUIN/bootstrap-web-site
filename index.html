Presentación

¡Hola! me llamo José Carlos Joaquín Vazquez.

Soy graudado de Ingeniería en Sistemas Computacionales del Instituto Tecnológico Superior de Jesús Carranza, en el estado de Veracruz, México.

Actualmente estoy enfocado en el desarrollo web, aprendiendo y aplicando tecnologias las cuales aprendo por cuenta propia, pues me gusta ser autodidacta.

Este sitio web personal lo he desarrollado utilizando las siguientes tecnologías:

  • HTML 5

    Para realizar el maquetado del sitio.

  • JAVASCRIPT

    Para ponerle funcionalidad al sitio: por ejemplo el formulario que podrás encontrar en el apartado de contacto.

  • PHP

    Para darle funcionalidad a la página.

  • CSS 3

    Para darle estilo a la página.

  • BOOTSTRAP 4

    Para utilizar componentes que generen un a buena estructura y vista del sitio.
